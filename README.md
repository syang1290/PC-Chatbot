# PC-Chatbot
RigCheck AI is a locally-hosted, privacy-first hardware appraisal tool designed to help users navigate the volatile used gaming PC market. Built using the Mistral-7B-Instruct architecture and served through a modern Panel web interface, the application leverages advanced prompt engineering to act as a veteran hardware specialist. It provides realistic "Quick Sell" and "Market Value" estimates for components like CPUs, GPUs, and RAM, delivering these insights in structured Markdown tables. By utilizing Python 3.13 and the ctransformers library—optimized for hardware acceleration on macOS via Metal—RigCheck AI offers a powerful, entirely offline experience that not only values builds but also identifies missing specifications and suggests logical upgrade paths for 2025 standards.
